install_rnaseq.sh - install several useful bioinformatics tools for BRB-SeqTools

* Bcftools
* Bowtie2
* BWA
* FastQC
* fastx
* HTSeq
* Picard-tools
* Samtools
* SnpEff
* SRAtoolkit
* STAR
* Tophat

and

* avfs for mounting a tarball
* Java 8 (from Sun or OpenJDK)
* Latex
* lftp 
* parallel
* python and pip
* PANDOC
* R including the rmakrdown package
